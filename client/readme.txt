Here are the resources to be served to the client from a separate web server. I guess you could technically run the web server from the pi, but this wasn't what I was going for. 

Hovering over the left and right eye shows the high score for the day and what the last score was. 

The socket.io.js is directly from socket.io, which is the medium for the web socket. 

eyeSocket.js is the web socket that connects to the pi. This retrieves the data from the pi in almost real time.
